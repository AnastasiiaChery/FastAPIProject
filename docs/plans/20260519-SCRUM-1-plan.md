# Task 1 — Basic FastAPI Health Check Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a FastAPI application with a `/health` endpoint that returns `{"status": "ok"}`, providing a working HTTP server foundation for the project.

**Architecture:** Create an `app.py` module exporting a FastAPI instance with a single health-check route. Keep `main.py` (AI PR Autopilot) untouched — the FastAPI app lives alongside it as the actual web service layer. Run with `uvicorn app:app`.

**Tech Stack:** Python 3.9+, FastAPI, uvicorn (already in `pyproject.toml`)

### Task 1: Create the FastAPI application module

**Files:**
- Create: `app.py`
- Create: `tests/__init__.py`
- Create: `tests/conftest.py`
- Create: `tests/test_app.py`

**Step 1: Write the failing test**

Create `tests/test_app.py`:

```python
from fastapi.testclient import TestClient
from app import app

client = TestClient(app)


def test_health_returns_200():
    response = client.get("/health")
    assert response.status_code == 200


def test_health_returns_ok_body():
    response = client.get("/health")
    assert response.json() == {"status": "ok"}


def test_health_content_type_is_json():
    response = client.get("/health")
    assert "application/json" in response.headers["content-type"]
```

**Step 2: Run test to verify it fails**

```bash
python -m pytest tests/test_app.py -v
```

Expected: `FAILED` — `ModuleNotFoundError: No module named 'app'`

**Step 3: Write minimal implementation**

Create `app.py`:

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/health")
def health() -> dict:
    return {"status": "ok"}
```

Create `tests/__init__.py` (empty).

Create `tests/conftest.py` (empty — placeholder for future fixtures).

**Step 4: Install test dependency and run tests**

```bash
pip install httpx
python -m pytest tests/test_app.py -v
```

Expected: All 3 tests PASS.
